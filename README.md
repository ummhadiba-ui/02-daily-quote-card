# 🌟 Daily Quote Card

A simple and beautiful motivational quote card built using HTML, CSS, and JavaScript.

## 📌 Features

- Motivational quote
- Author name
- Beautiful card design
- Circular image
- Hover effect on button
- JavaScript alert
- Console message

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Use

1. Open `index.html` in your browser.
2. Click the **Inspire Me** button.
3. An alert message will appear.
4. Open the browser console (F12 → Console) to see the JavaScript message.
